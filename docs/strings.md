# 線種 (strings)

- 考量什麼：**經濟**？**打感**？**細彈**？
- 不同的球友、不同的打法、單雙打 會喜歡不同的 **球線摩擦感**
  * 塗層摩擦感 e.g. 66UM
  * 編織摩擦感 e.g. BG80
  * 金屬塗層（例如鈦 Ti 塗層）聲音清亮 會比較滑

## **該怎麼選線種？**

常見線徑：<- 0.65mm <-> 0.68mm <-> 0.70mm ->

- 經濟考量：BG65
- 打感考量：BG80
- 細彈考量：BG66UM、BGAS
- 特殊考量：BGAB（豎線、橫線 不同）

## 大盤線

- 常見 長度 200m 可以穿到 20+1~4 支
- 量少 少穿 線會放太久 變質問題
