# **阿達力 阿達利 阿搭力 當たり atari**

- **阿達力** **阿達利** **阿搭力** 是日文「**当たり**」**あたり**（**atari**） 基本上的意思就是「**命中**」「**打中**」「**中獎**」「**成功**」 
- 在日本吃冰棒，如果吃到 **再來一支** 冰棒上面也會印 **当たり** **あたり** **atari** 囉！
- 棒球 打者確實 打中 **球棒甜蜜點 sweet spot** 有「**咬到球**」也會說成 **打到阿達力** atari

## **擊球聲不似別人那樣響亮**

- 直覺會以為要打在拍面「**正中央**」擊球聲反而 **較悶**
- 打在 **正中央偏上方** 一兩條線 擊球聲明顯 比較 **響亮**
- 中央偏上方 打到 **甜蜜點** 時 **最不震手！** **衝擊力最大！**
