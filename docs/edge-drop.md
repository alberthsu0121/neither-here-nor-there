# **倒拍關鍵 - Edge Drop**

**Swing** **不夠大** **不夠漂亮**，建議檢視 **倒拍** 是否 **夠到位**！？

## **先聽聽專業的怎麼說「倒拍」**

- 球拍會自己的「**放下來**」「**倒下來**」**（真的假的！）**

    * 沒有矯枉過正的做 做到 **肌肉記憶** 的話，其實 **拍頭** **不會**「**自己倒下來**」

- 長球最重要的是這個「**把倒拍做好**」

    * **漂亮的倒拍** 是 **高吊殺** 不可或缺 的 一部分

- 很多人 **放不下來「倒拍倒不下來」** 尤其是 **業餘的大人**

    - **練家子小朋友** 也不是每一位都 **倒拍倒得很漂亮**；**沒倒好的也陣亡了**，**我們通常只看到倒好的**
    - **業餘的大人** **還能存活是因為靠成年人的力量 + tiny swing 揮擊** **此舉也封印限制球技成長的天花板高度**

## **把拍倒好的重點在於方法（怎麼順你的拍頭軌跡）**

- **動態框架** 式的 **架拍** 動作：**側身** **走肩** **抬頭** **看球頭**

    - **吊腕** 會 更容易銜接 **立腕 這是 Edge Drop 的起始點**

![**動態框架** 後視](dynamic.frame.structure.backview.png) ![**動態框架** 側視](dynamic.frame.structure.sideview.png)

- **Edge Drop** 精髓在於球拍的刀面 (Edge) 是以類似 **梳頭** 的動作 **往身後倒拍**
    - 拍頭 Ｔ點 中桿 **非常靠近額頭但不會打到額頭**
    - **像梳慣用側的額頭頭髮 由前往後梳 這就是 Edge Drop 經過額頭的時候**
    - **手腕處於吊腕 <-> 立腕 轉換期**

![**Edge Drop 梳頭** 後視](edge.drop.brush.backview.png) ![**Edge Drop 梳頭** 側視](edge.drop.brush.sideview.png)

- **此消彼長** **拍頭向下** **肘就會向上**
    - **像梳慣用側的額頭頭髮 由前往後梳 這就是 Edge Drop 梳頭到底的時候**
    - **手腕是立腕狀態**
    - **Edge Drop 到最底下 -> **向慣用外側撇拍** -> 向上銜接出拍；形成一個不停歇的橢圓軌跡**
    - **有點像賽車 過髮夾彎 不會有中間的停頓 這樣 swing 才會又大又漂亮**

![**此消彼長 拍頭向下 肘就會向上** 後視](racket.head.drop.elbow.up.backview.png) ![**此消彼長 拍頭向下 肘就會向上** 側視](racket.head.drop.elbow.up.sideview.png)

- **斜肩** 促成 **肘高於耳** **拍頭外撇** 銜接 **向上伸展**
- 這個是 **向慣用外側撇拍** 時機點

![**斜肩** 促成 **肘高於耳** **拍頭外撇** 銜接 **向上伸展** 後視](tilt.shoulder.elbow.upwards.ear.backview.png) ![**斜肩** 促成 **肘高於耳** **拍頭外撇** 銜接 **向上伸展** 側視](tilt.shoulder.elbow.upwards.ear.sideview.png)
